Test procedure for Blockchain based Member Management System

Required softwares

testrpc
Web browser
*node is not necessary

Steps

Uncompress zip file
Install web3 module. ex) npm install web3
Start testrpc
Modify path to web3 module in index.html file
Open index.html with Web browser
Click 'Choose File' button on webpage
Choose member.sol file
Wait until Contract Address shows up on top of webpage
Try to add new member
Try to delete existing member
